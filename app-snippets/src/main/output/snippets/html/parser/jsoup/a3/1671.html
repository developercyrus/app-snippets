創業通 : 西日合璧　街頭炸物店月賺2萬

文章日期：2009-10-05

做生意好講創意，經營食肆也不例外，不加多一點新意，就很難立足，即使獨沽一味賣炸物，都可以千變萬化，像剛開業4個月的嵐山料理，老闆阿嵐和Edwin均是廚師出身，認為街頭小食，只要充滿創意，不難殺出血路。他們懂得把西式的材料，如芝士、鵝肝等，結合日式的做法，做出超過20多款不同的炸物，日賣逾百盒，成功於短短2個月內轉虧為盈，吸納到一班熟客支持，月賺2萬。
位於銅鑼灣中央圖書館旁的嵐山料理，店鋪面積雖只得70呎，在採訪當天，週末下午4時，圍�一大堆客人，正品嘗�新鮮即炸的食物，其中客人周小姐是第一次幫襯，「之前睇到雜誌介紹，覺得�食物好正，想�試�，今次試�炸芝士，真係好好味，價錢ok唔算太貴，下次我會再�試其他食物。」
嵐山料理主要賣各式炸物，但並非普通炸番薯、天婦羅，而是把西式和日式的食物和製法結合，如炸芝士、炸生蠔、炸鵝肝薯茸丸、炸皇帝蕉等，全部即叫即炸，保證新鮮。想到賣這些特色炸物的，是28歲的老闆阿嵐和30多歲的拍檔Edwin，二人年紀不大，卻在飲食業已有超過10年經驗，「我�兩個都係17歲入行，做過好多唔同類型�餐廳，好似日本料理、Cova等。」
小本創業炸物招徠
一直希望開鋪創業的他們，資金有限，只能投資20萬開設街頭小食店，但也絕不馬虎。「雖然創業�錢只夠開小食店，但我�唔想做魚蛋、雞蛋仔呢�咁簡單�小食，由於我�之前�日本餐廳做過，覺得香港人普遍都鍾意食日本�，而且做小食最�做熱辣辣�食物，就諗到做類似天婦羅�炸物，為求特別�，揀�西式�材料，希望畀到客人新鮮感。」於是他們想到採用炸芝士、炸鵝肝芝心薯茸丸等食物，結果打響了名堂。
別看現時店鋪堆滿人，原來開業初期曾遇上不少困難，導致生意慘淡，開業首月便蝕錢，「我�住�港島區，以前返工成日經過圖書館呢頭，發現下午人流都幾多，而且附近有唔少學校，小食又�學生胃口，所以揀��呢度開業。」他們找來一條龍的裝修和�牌照公司幫忙，由於沒有簽約，只得口頭承諾，結果該公司遲了2星期完工，要他們白交半個月租金，「我�冇做過生意、又冇簽約，唔識去追討賠償，白白��半個月租。」
豬流感修路冇生意
店內裝修好，又輪到店外維修公路，四周圍起欄，維修了近2個星期，嚴重影響生意。「出面維修公路又嘈又沙塵滾滾，�人行過，都唔願入�幫襯啦。」加上原本計劃做學校生意，誰知今年6月一開業，學校便因豬流感而停課，真是禍不單行。「我�預�要守，但冇諗過一開鋪就有咁多問題。」
幸好他們並不灰心，想盡辦法吸引客人。「冇客幫襯，惟有自己搵客，我��住�食物，�鋪頭附近不斷請人試食，想睇�客人反應之餘，亦希望佢�覺得好食，會再�幫襯。」此外，他們會於繁忙時間，在附近巴士站派傳單，又會等圖書館關門時，於門外繼續派，甚至推出送外賣服務，「開頭冇乜生意，咩方法都要試�，覺得送外賣可以方便到客人，就提供埋外賣服務，鋪頭附近�客只要買滿$50，就可以送外賣，而天后及銅鑼灣就$100，其他區就冇辦法送貨，因為�食物唔可以�到咁遠，會唔新鮮。」
經過1個月不斷努力，店鋪終於吸納到一班熟客之餘，還吸引了不少傳媒採訪，「班熟客好好，自願幫我��附近派傳單，有�教會朋友仲�番去派，之前我�見生意咁差，都好灰心o架，唔知可以捱到幾耐，但係見到�熟客同朋友咁幫手，都鼓勵到我�唔可以咁容易放棄，後來有傳媒�做訪問，真係好鼓舞，好似得到認同咁，先有信心繼續做落去。」
新款炸物保新鮮感
由於他們採用的食材較貴，例如選用法國金文不芝士、鵝肝等，成本較高，故每款炸物售價一般也要$20以上，他們擔心客人嫌貴，便把最好賣的招牌菜炸芝士，定為特價貨，由當初$28降為$22，「我�最初都係跟番之前做開�餐廳，訂貨�供應商去訂貨，所以來貨價比較貴，但我�又唔敢賣太貴，所以賺得好少，毛利唔夠3成。好彩開�鋪之後，有好多食品公司會主動�sell貨，品質相若�食材，價錢可以平2至3成，經營成本平�，o依家毛利約有一半左右。」
開業第2個月，他們已轉虧為盈，每日賣出過百盒炸物，月賺近2萬，有時週末、週日，客人更要在店外等候，「食物即叫即炸，平均要用3分鐘時間，有時人多，可能要等耐�，但�客見你即炸，反而唔介意等，覺得新鮮炸起會好食好多。」
現時嵐山料理共有超過20款食品供選擇，以炸芝士、炸生蠔最熱賣，為求保持新鮮感，會不停推出新食物應市，像開業時正值炎夏之時，加上賣炸物，感覺熱上加熱，於是他們亦加入較為消暑的食品，如特色凍飲和冷麵等，「希望�客食炸�時，覺得好熱，可以買杯�飲，或者食個冷麵中和番。」
他們更每星期推出2至3款新產品，如炸軟殼蟹、炸芝腿卷等，「我�一得閒就會研究新產品，跟住試做同搵朋友試食，覺得ok就會推出，而一�唔賣得�食物，好似之前有款炭燒雞串冇咁好賣，我�就會暫停。」除了食物，連配搭的醬料也有6款選擇，如黑加侖子汁、辣蟹籽沙律醬、鵝肝沙律醬、芥辣醬、威士忌醬及他他汁，也是由他們自己研製，「通常我�會幫客人配搭好醬料，好似食炸芝士要配威士忌醬，不過客人可以自己揀�咩組合。」
搬大鋪增存貨空間
暑假過去，學校再開學，對他們來說也是一大喜訊，可以擴大客源，「我�增加�學生餐，$18可以�5款指定食物中揀1款，再搭埋飲品，幾受學生歡迎。」像就讀中學的阿Joe，每星期便至少來幫襯2至3次，「我�附近返學，試過佢���食，覺得好好味，而且佢�成日有新�出，所以我個個星期都�試新�。」開學後，店鋪的生意也增加了1至2成。現時以年輕客如學生、OL最多，佔8成，其餘為街坊客。
雖然開業只得4個月，但阿嵐和Edwin已計劃搬鋪，希望找一間較大的鋪位，做多點生意，「呢間鋪真係好細，o依家生意好�，根本冇位存貨，有時未到下晝，某幾樣食物已經賣晒，搞到�客人白等，真係唔好意思，所以好想�附近搵間大��鋪，令存貨量增加，我�可以再推出多�產品。」

邊度入貨？
雖然Edwin和阿嵐已在飲食界打滾多年，但是他們指要入平、靚、正的貨品，都要不停搵不同的店鋪�貨和格價，「好似我�最初跟以前做過�鋪頭入貨，品質雖然好，但價錢會較貴，之後我�都會自己�黃頁，或者跟住貨品上面寫住�供應商聯絡方法，打電話去問�佢�仲有咩貨、大約幾多錢等等，如果覺得�用，可以先買少少試�，品質好先向佢�訂貨，不過�細鋪開頭訂貨量少，未必可以�到優惠或折扣，但只要訂多幾次貨，等對方覺得大家係長期合作關係，就可以嘗試傾�折扣。」
裝修逾期有得追
相信不少人都在創業過程中，會遇到像阿嵐和Edwin的情況，因為裝修逾期，導致白交2星期租金，雖然他們沒有與裝修公司簽約，但其實一樣可以作出追討。梁永鏗律師表示，若裝修公司曾口頭承諾一個完工日子，便具有法律效用，「�法律上，只要對方有口頭承諾，一樣可以去追討損失，首先原告人要搵會計師計好損失�數目，如果少過5萬就向小額錢債審裁處申請，多過5萬就係區域法院，雖然大家冇簽合約，但法官會根據雙方�口供、裝修時間長短，中間有冇改動過裝修等因素，�作出裁決。」
